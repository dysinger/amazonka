Description

Returns the scaling activities for the specified Auto Scaling group. If the specified ActivityIds list is empty, all the activities from the past six weeks are returned. Activities are sorted by the start time. Activities still in progress appear first on the list. This action supports pagination. If the response includes a token, there are more records available.
