Network.AWS.CloudWatchLogs.V2014_03_28.GetLogEvents

Description

Retrieves log events from the specified log stream. You can provide an optional time range to filter the results on the event timestamp. By default, this operation returns as much log events as can fit in a response size of 1MB, up to 10,000 log events. The response will always include a nextForwardToken and a nextBackwardToken in the response body. You can use any of these tokens in subsequent GetLogEvents requests to paginate through events in either forward or backward direction. You can also limit the number of log events returned in the response by specifying the limit parameter in the request.
