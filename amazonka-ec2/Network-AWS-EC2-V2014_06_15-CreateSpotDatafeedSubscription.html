Network.AWS.EC2.V2014_06_15.CreateSpotDatafeedSubscription

Description

Creates a datafeed for Spot Instances, enabling you to view Spot Instance usage logs. You can create one data feed per AWS account. For more information, see Spot Instances in the Amazon Elastic Compute Cloud User Guide.
