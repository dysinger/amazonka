Description

Describes one or more of your bundling tasks. Completed bundle tasks are listed for only a limited time. If your bundle task is no longer in the list, you can still register an AMI from it. Just use RegisterImage with the Amazon S3 bucket name and image manifest name you provided to the bundle task. Example 1 This example describes the status of the specified bundle task. https:/ec2.amazonaws.com?Action=DescribeBundleTasks &amp;bundleId.1=bun-c1a540a8 &amp;AUTHPARAMS 59dbff89-35bd-4eac-99ed-be587EXAMPLE i-12345678 bun-c1a540a8 cancelling 2008-10-07T11:41:50.000Z 2008-10-07T11:51:50.000Z myawsbucket winami 20% Example 2 This example filters the response to include only bundle tasks whose state is either complete or failed, and in addition are targeted for the Amazon S3 bucket named myawsbucket. https:/ec2.amazonaws.com?Action=DescribeBundleTasks &amp;Filter.1.Name=s3-bucket &amp;Filter.1.Value.1=myawsbucket &amp;Filter.2.Name=state &amp;Filter.2.Name.1=complete &amp;Filter.2.Name.2=failed &amp;AUTHPARAMS.
