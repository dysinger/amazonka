Network.AWS.EC2.V2014_06_15.DescribeVolumeAttribute

Description

Describes the specified attribute of the specified volume. You can specify only one attribute at a time. For more information about Amazon EBS volumes, see Amazon EBS Volumes in the Amazon Elastic Compute Cloud User Guide.
