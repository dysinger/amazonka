Description

Resets an attribute of an AMI to its default value. Example This example resets the launchPermission attribute for the specified AMI. https:/ec2.amazonaws.com?Action=ResetImageAttribute &amp;ImageId=ami-61a54008 &amp;Attribute=launchPermission &amp;AUTHPARAMS &lt;ResetImageAttributeResponse xmlns="http://ec2.amazonaws.com/doc/2014-02-01/"&gt; &lt;requestId&gt;59dbff89-35bd-4eac-99ed-be587EXAMPLE&lt;/requestId&gt; &lt;return&gt;true&lt;return&gt; &lt;ResetImageAttributeResponse&gt;.
