Description

Lists the MFA devices. If the request includes the user name, then this action lists all the MFA devices associated with the specified user name. If you do not specify a user name, IAM determines the user name implicitly based on the AWS access key ID signing the request. You can paginate the results using the MaxItems and Marker parameters. https:/iam.amazonaws.com ?Action=ListMFADevices &UserName=Bob &AUTHPARAMS Bob R1234 false 7a62c49f-347e-4fc4-9331-6e8eEXAMPLE.
