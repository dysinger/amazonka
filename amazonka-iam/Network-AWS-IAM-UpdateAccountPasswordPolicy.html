Description

Updates the password policy settings for the account. For more information about using a password policy, see Managing an IAM Password Policy in the Using IAM guide. https:/iam.amazonaws.com ?Action=UpdateAccountPasswordPolicy &MinimumPasswordLength=12 &RequireSymbols=false &RequireNumbers=true &RequireUppercaseCharacters=true &RequireLowercaseCharacters=true &AllowUsersToChangePassword=true &MaxPasswordAge=90 &PasswordReusePrevention=6 &HardExpiry=false &Version=2010-05-08 &AUTHPARAMS 7a62c49f-347e-4fc4-9331-6e8eEXAMPLE.
