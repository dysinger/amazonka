Description

Retrieves the specified policy document for the specified user. The returned policy is URL-encoded according to RFC 3986. For more information about RFC 3986, go to http:/www.faqs.orgrfcs/rfc3986.html. https:/iam.amazonaws.com ?Action=GetUserPolicy &UserName=Bob &PolicyName=AllAccessPolicy &AUTHPARAMS Bob AllAccessPolicy {Version:"2012-10-17",Statement:[{Effect:Allow,Action:"*",Resource:"*"}]} 7a62c49f-347e-4fc4-9331-6e8eEXAMPLE.
