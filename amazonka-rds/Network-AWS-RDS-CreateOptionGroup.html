Description

Creates a new option group. You can create up to 20 option groups. https:/rds.amazonaws.com ?Action=CreateOptionGroup &OptionGroupName=myoptiongroup &EngineName=oracle-se1 &MajorEngineVersion=11.2 &OptionGroupDescription=Test option group 11.2 myoptiongroup oracle-se1 Test option group b2416a8a-84c9-11e1-a264-0b23c28bc344.
