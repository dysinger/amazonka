Network.AWS.Redshift.V2012_12_01.RevokeSnapshotAccess

Description

Removes the ability of the specified AWS customer account to restore the specified snapshot. If the account is currently restoring the snapshot, the restore will run to completion. For more information about working with snapshots, go to Amazon Redshift Snapshots in the Amazon Redshift Management Guide.
