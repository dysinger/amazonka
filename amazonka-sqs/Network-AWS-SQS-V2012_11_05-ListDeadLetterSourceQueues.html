Description

Returns a list of your queues that have the RedrivePolicy queue attribute configured with a dead letter queue. The following example Query request returns a list of dead letter source queues. In this example only one source queue, MySourceQueue, was configured with a dead letter queue. Action=ListDeadLetterSourceQueues &Version=2012-11-05 http:/sqs.us-east-1.amazonaws.com123456789012/MySourceQueue 8ffb921f-b85e-53d9-abcf-d8d0057f38fc For more information about using dead letter queues, see Using Amazon SQS Dead Letter Queues.
