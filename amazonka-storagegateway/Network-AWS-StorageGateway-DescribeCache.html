Description

This operation returns information about the cache of a gateway. This operation is supported only for the gateway-cached volume architecture. The response includes disk IDs that are configured as cache, and it includes the amount of cache allocated and used. Example Request The following example shows a request to obtain a description of a gateway's working storage.
