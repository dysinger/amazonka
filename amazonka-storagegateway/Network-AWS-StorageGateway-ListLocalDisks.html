Description

This operation returns a list of the local disks of a gateway. To specify which gateway to describe you use the Amazon Resource Name (ARN) of the gateway in the body of the request. The request returns all disks, specifying which are configured as working storage, stored volume or not configured at all. Example Request The following example shows a request that returns information about a gateway's local disks.
