Network.AWS.StorageGateway.V2013_06_30.DescribeCache

Description

This operation returns information about the cache of a gateway. This operation is supported only for the gateway-cached volume architecture. The response includes disk IDs that are configured as cache, and it includes the amount of cache allocated and used.
