Network.AWS.StorageGateway.V2013_06_30.DescribeSnapshotSchedule

Description

This operation describes the snapshot schedule for the specified gateway volume. The snapshot schedule information includes intervals at which snapshots are automatically initiated on the volume. Example Request The following example shows a request that retrieves the snapshot schedule for a volume.
