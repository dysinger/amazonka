Network.AWS.CloudTrail.V2013_11_01.CreateTrail

Description

From the command line, use create-subscription. Creates a trail that specifies the settings for delivery of log data to an Amazon S3 bucket. Support for passing Trail as a parameter ends as early as February 25, 2014. The request and response examples in this topic show the use of parameters as well as a Trail object. Until Trail is removed, you can use either Trail or the parameter list.
