Network.AWS.EC2.V2014_06_15.BundleInstance

Description

Bundles an Amazon instance store-backed Windows instance. During bundling, only the root device volume (C:) is bundled. Data on other instance store volumes is not preserved. This procedure is not applicable for Linux/Unix instances or Windows instances that are backed by Amazon EBS. For more information, see Creating an Instance Store-Backed Windows AMI.
