Network.AWS.EC2.V2014_06_15.DeregisterImage

Description

Deregisters the specified AMI. After you deregister an AMI, it can't be used to launch new instances. This command does not delete the AMI. Example This example request deregisters the specified AMI. https:/ec2.amazonaws.com?Action=DeregisterImage &amp;ImageId=ami-4fa54026 &amp;AUTHPARAMS &lt;DeregisterImageResponse xmlns="http://ec2.amazonaws.com/doc/2014-02-01/"&gt; &lt;requestId&gt;59dbff89-35bd-4eac-99ed-be587EXAMPLE&lt;/requestId&gt; &lt;return&gt;true&lt;return&gt; &lt;DeregisterImageResponse&gt;.
