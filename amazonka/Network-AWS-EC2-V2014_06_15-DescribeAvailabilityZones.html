Network.AWS.EC2.V2014_06_15.DescribeAvailabilityZones

Description

Describes one or more of the Availability Zones that are available to you. The results include zones only for the region you're currently using. If there is an event impacting an Availability Zone, you can use this request to view the state and any provided message for that Availability Zone. For more information, see Regions and Availability Zones in the Amazon Elastic Compute Cloud User Guide. Example This example request describes the Availability Zones that are available to you.
