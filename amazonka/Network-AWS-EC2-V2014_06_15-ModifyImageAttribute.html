Network.AWS.EC2.V2014_06_15.ModifyImageAttribute

Description

Modifies the specified attribute of the specified AMI. You can specify only one attribute at a time. AWS Marketplace product codes cannot be modified. Images with an AWS Marketplace product code cannot be made public. Example.
