Network.AWS.ElasticBeanstalk.V2010_12_01.RequestEnvironmentInfo

Description

Initiates a request to compile the specified type of information of the deployed environment. Setting the InfoType to tail compiles the last lines from the application server log files of every Amazon EC2 instance in your environment. Use RetrieveEnvironmentInfo to access the compiled information.
