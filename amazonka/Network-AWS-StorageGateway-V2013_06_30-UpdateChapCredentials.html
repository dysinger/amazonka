Network.AWS.StorageGateway.V2013_06_30.UpdateChapCredentials

Description

This operation updates the Challenge-Handshake Authentication Protocol (CHAP) credentials for a specified iSCSI target. By default, a gateway does not have CHAP enabled; however, for added security, you might use it. When you update CHAP credentials, all existing connections on the target are closed and initiators must reconnect with the new credentials. Example Request The following example shows a request that updates CHAP credentials for an iSCSI target.
